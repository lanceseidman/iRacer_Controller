THIS PROJECT IS TO HELP WP8 USERS INTERACT HIS/HER i-Racer WITH A WINDOWS PHONE 8 DEVICE USING BLUETOOTH.

WHAT CAN THE CODE DO NOW?

** CONNECT/ESTABLISH BLUETOOTH CONNECTION

** SEND UP/DOWN/RIGHT/LEFT COMMANDS FOR CONTROL

WHAT'S NEXT/MUST BE DONE?

** SENSORS: IR (TVBGONE), ULTRASONIC, GPS

** CAMERA STREAM

** AUDIO RECORDING
